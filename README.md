# Parcial – Autómatas y Procesamiento de Lenguajes
1. **Punto 1 – AFD para movimientos de ajedrez**
   Se definió una expresión regular y se implementó un autómata finito determinista para validar movimientos de ajedrez a partir de cadenas de entrada.

2. **Punto 2 – AFD para reconocimiento de identificadores**
   Se diseñó un autómata finito para reconocer identificadores válidos que comienzan con una letra y pueden continuar con letras o números.

3. **Punto 3 – Método de Newton**
   Se implementó el método de Newton-Raphson para aproximar la raíz de una función, mostrando las iteraciones hasta alcanzar la convergencia.

4. **Punto 4 – Algoritmo de Euclides**
   Se desarrolló una implementación del algoritmo de Euclides para calcular el máximo común divisor (MCD) entre dos números.

5. **Punto 5 – Serie de Maclaurin con ANTLR**
   Se utilizó ANTLR para reconocer una expresión de entrada y calcular una aproximación de (e^x) utilizando la serie de Maclaurin.
